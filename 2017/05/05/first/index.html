搭建HEXO

虽然搭建一个简单的HEXO并不难，但是我还是尝试写一篇简单的日志，学习了这么久第一次开始总结自己的经历。

准备

环境安装

  1. node.js

    因为我们使用的 HEXO 需要 node.js 以及 npm 包管理工具的支持。
    所以我们需要去 node.js 的官网下载 node 环境。下载

  2. HEXO

    安装 HEXO 的步骤为在CMD里面执行

    npm install hexo-server –save

    cnpm install hexo-server –save

    npm install hexo-deployer-git -g

    deploy:

    type: git

repo: git@github.com:yourname/yourname.github.io.git

branch: master

    ```

  3. 每次在 hexo generate 之后我们就可以使用 hexo deploy 去发布我们的 hexo 博客。

  4. 至此我们的博客就能搭建完成了，报错的话我会另起一篇博客。

hexo 安装排错

这里记载了部分我们可能遇见的 hexo 搭建中的错误。

  • hexo server 命令没有反应
  • “cannot get / “
  • hexo deploy 命令没有反应
  • hexo deploy 命令出现问题

hexo server 命令没有反应

　　这个问题是因为没有安装 hexo-server 模块导致的。

“cannot get / “

　　这个问题是没有安装 hexo 所需的模块导致的。在 hexo 博客的文件夹使用以下命令。

hexo deploy 命令没有反应

　　这个问题是因为没有安装 hexo-server 模块导致的。

hexo deploy 命令出现问题

　　这个问题分很多情况，可能出自于 git 客户端设置账号的时候并没有设置好 ssh-key 导致无法上传。参考：添加SSH key

　　另外一个原因就是 yml 设置文件的时候，你使用的 repo 不是 git@github.com:yourname/yourname.github.io.git 形式。
